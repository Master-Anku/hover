အခန်း (၁)

HTML

HTML ဆိုတာ Hyper Text Markup Language တစ်ခုပါ။

HTML ကို ဘယ်လိုရေးသလဲ။

HTMLကို ရေးတဲ့ အခါမှာ
၁။ ခေါင်းစဉ်(header)
၂။ ဘော်ဒီ (body)
၃။ ခြေထောက် (footer)ဆိုပီး စနစ်တကျ ဖြစ်အောင် အပိုင်း(၃)ပိုင်း ခွဲပီး ရေးရပါတယ်။
ရေးသားတဲ့ နေရာမှာ body အဖွင့် အပိတ်ထဲမှာဘဲ အပိုင်း(၃)ပိုင်းခွဲပီး ရေးသားရမှာပါ။

ဒီအပိုင်းမှာတော့ headerရေးနည်းကိုဘဲ အဓိက ထားပီး ရေးသားတင်ပြပေးပါမယ်။
header ဆိုတာ ဘာလဲ။
web size တစ်ခုရဲ့ ခေါင်းစဉ်ကို headerလို့ ခေါ်ပါတယ်။
ခေါင်းစဉ်မှာ logo image menuBar စသည်ဖြင့် ပါဝင်ပါတယ်။
headerရေးနည်းကတော့
ပထမဦးဆုံး html ထဲက body အဖွင့်အပိတ်ထဲမှာ အောက်ပါ ပုံမှာပြထားသလို ရေးကြည့်ပါ။

ပုံမှာတွေ့ရတဲ့ အစိမ်းရောင် စာတန်းလေးကတော့ commit ရေးထားတာပါ။
ကုဒ်ကို ရေးပီးတဲ့အခါမှာ "save"ပီး "Browser"မှာ ကြည့်တဲ့အခါ အောက်ပါပုံလို ဖြစ်နေတာကို တွေ့ရမှာပါ။
